# Bike-Purchase-Analysis
*Project Objective*
The goal of this project was to analyze customer demographic data for a bike retail business to identify key drivers behind bike purchases. By understanding these patterns, the business can better target its marketing efforts toward high-conversion segments.
# Tools Used
Microsoft Excel: Data Cleaning, Pivot Tables, and Dashboarding.
Data Visualization: Interactive Slicers and Charts.

# *Data Cleaning & Transformation*
Before analysis, I performed the following steps to ensure data integrity:
Handling Duplicates: Identified and removed 26 duplicate records.
Standardization: Replaced abbreviations (e.g., "M" to "Married," "F" to "Female") for better readability.
Feature Engineering: Created an Age Range column using a nested IF statement to group customers into:
Adolescent (<31)
Middle Age (31-54)
Unc (55+)
Key Insights
After analyzing the 1,000+ customer records, three major trends emerged:
Income Levels: Customers who purchased bikes had a significantly higher average income ($55k+) compared to non-purchasers ($46k).
Commute Distance: The highest conversion rate was found among customers living within 0-1 miles of the store.
Demographics: Middle-aged professionals were the most likely segment to invest in a new bicycle
